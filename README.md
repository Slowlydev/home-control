# home-control
A nextron / electron app to control your hue smarthome.

Made with:

- nextron
- electron
- nextjs
- framer-motion
- swr
- axios

## Setup

### Install Dependencies

```
# open repository
$ cd home-control

# install dependencies with yarn
$ yarn
```

### Use it

```
# development mode
$ yarn dev

# production build
$ yarn build 
```
